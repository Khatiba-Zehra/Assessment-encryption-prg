Python solution for decoding encrypted messages.
Prints all valid decodings in
alphabetical order and uses brute force. 
Mapping for alphabets are given below :-
A = ._     B = _...   C = _._.   D = _..
E = .      F = .._.   G = __.    H = ....
I = ..     J = .___   K = _._    L = ._..
M = __     N = _.     O = ___    P = .__.
Q = __._   R = ._.    S = ...    T = _
U = .._    V = ..._   W = .__    X = _.._
Y = _.__   Z = __..
